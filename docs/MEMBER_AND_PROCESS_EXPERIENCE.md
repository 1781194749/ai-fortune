# 会员机制与问事流程体验

## 会员机制口径

- 天：轻入口和留存。免费版与轻体验只承担建档、每日轻问事、每日塔罗等低门槛动作。
- 周：复盘点。周复盘用于整理一周内反复出现的问题、变化信号和下一步追问。
- 月：陪伴期。¥29 月度陪伴用于 30 天持续问事、档案记忆、必要追问余量和少量报告/手相额度。
- 关键阶段：¥99 方案是一个核心主题的 30 天阶段陪伴包，不是更大的星力包。它包含主题锁定、每 7 天复盘、30 天阶段总结和必要推演额度。

## 问事流程标准

每次正式问事或工具推演都应该按四段呈现：

1. 发送前：说明服务类型、推演方式、预计消耗和交付物。
2. 生成中：展示真实步骤，例如识别主题、读取档案、选择工具、生成结论。
3. 回答后：展示结论卡，并给出可继续追问或查看报告的下一步。
4. 失败时：明确本次没有生成报告或污染聊天记录，保留用户输入并提供重试。

## 已落地范围

- Chat 发送前会识别问事类型、推荐推演方式，并允许切换快速问答、正式问事和深度推演。
- Chat 生成过程由后端 `data-chatProgress` 真实事件驱动，不再使用定时器模拟步骤。
- 塔罗逐张翻牌、八卦按本卦/动爻/变卦展开、八字先显示四柱与五行分布，再进入回答生成。
- Chat 回答后展示结构化结论卡、动态追问和深度报告入口；失败时提供重新生成、轻量回答和反馈。
- 塔罗、八字、八卦工具页已经接入统一流程面板。
- 三个工具接口在余额不足或扣减失败时返回服务化文案。
- 三个工具接口改为扣减成功后再创建报告，避免扣减失败留下报告记录。
- 价格页公开方案聚焦免费、¥29 月度陪伴、¥99 关键阶段陪伴。
