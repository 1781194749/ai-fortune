# 玄机 AI 项目目标规划

## 项目目标

玄机 AI 是一个面向国内优先、海外预留的 AI 命理陪伴网站。第一版目标是跑通从获客、体验、付费、报告沉淀到会员留存的完整闭环。

核心定位：

```text
看得见推演过程的 AI 命理顾问
```

## 当前已定

- 品牌：玄机 AI / Xuanji AI
- 首发：中文 Web 响应式，英文结构预留
- 风格：黑金、高级、神秘、可信
- 登录：邮箱验证码先可用，微信扫码登录预留
- 支付：支付宝、微信支付按正式系统设计，开发期走 mock payment
- 存储：七牛云
- AI：OpenAI Responses API
- 技术：Next.js、TypeScript、Tailwind CSS、PostgreSQL、Prisma

## 阶段计划

### 第一阶段：MVP 骨架

目标：形成可演示的网站和清晰的工程基础。

- 首页、价格页、核心功能入口
- 会员和星力值规则展示
- 文档、数据库模型、环境变量样例
- 模拟支付开关
- 七牛云和真实支付配置占位

### 第二阶段：核心体验

目标：让用户能完成一次真实体验。

- 邮箱验证码登录
- AI 对话会话
- 塔罗三牌阵
- 八字五行简析
- 手相上传流程
- 报告中心
- 星力值扣费

### 第三阶段：商业闭环

目标：按真实收费产品运行。

- 订单系统
- 支付宝电脑网站支付
- 微信 Native 扫码支付
- 支付回调校验
- 会员发放和星力值到账
- 轻后台查看用户、订单、报告和调用日志

### 第四阶段：增长与海外预留

目标：提升转化和复购。

- 分享报告图
- 邀请返星力
- 英文文案和 i18n
- Stripe 预留
- 今日运势和主题报告活动
